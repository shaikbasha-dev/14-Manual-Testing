# Manual Testing

## Overview

This repository contains comprehensive notes on Manual Testing, prepared as part of my journey toward becoming a Software Test Engineer and Java Full Stack Developer.

The content is organized in a structured manner, covering software testing fundamentals, Software Development Life Cycle (SDLC), Agile and Scrum, various testing types, Software Testing Life Cycle (STLC), Defect Management, and advanced testing concepts. Each topic includes detailed explanations, practical examples, workflows, diagrams, interview questions, and key points for better understanding.

This repository serves as a learning resource, interview preparation guide, and quick reference for students, freshers, and software professionals.

---

## Repository Objectives

- Learn the fundamentals of Manual Testing.
- Understand Software Development Life Cycle (SDLC).
- Study Agile Methodology and Scrum Framework.
- Learn different Software Testing Types.
- Understand Software Testing Life Cycle (STLC).
- Learn Defect Tracking and Bug Life Cycle.
- Explore advanced software testing concepts.
- Prepare for Manual Testing interviews.
- Build a structured knowledge repository for future reference.
