# Manual Testing

## Overview

This repository contains comprehensive notes on **Manual Testing**, prepared as part of my journey toward becoming a **Software Test Engineer** and **Java Full Stack Developer**.

The content is organized in a structured manner, covering **Software Testing Fundamentals**, **Software Development Life Cycle (SDLC)**, **Agile and Scrum**, various **Testing Types**, **Software Testing Life Cycle (STLC)**, **Defect Management**, and **Advanced Testing Concepts**.

Each topic includes detailed explanations, practical examples, workflows, comparison tables, diagrams, real-time scenarios, interview questions, and key points for better understanding.

In addition, this repository contains a dedicated **Manual Testing Interview Questions and Answers** section featuring **100 carefully selected interview questions with detailed answers**, designed to help students, freshers, QA Engineers, and software professionals prepare confidently for technical interviews.

This repository serves as a learning resource, interview preparation guide, and quick reference for students, freshers, and software professionals.
